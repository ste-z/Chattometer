How This is Calculated?

We uses the ecologits package to calculate environmental impacts. Here, we provide a overview of the method. Please refer to the ecologits API reference for more information.

Calculation of Impacts

The impacts are further divided into:

Usage Impact

This refers to the energy consumption of the server during inference. The calculation involves:

  • Single GPU usage modeled using linear regression trained on Hugging Face’s LLM Leaderboard dataset, relating GPU energy per output token to the model’s active parameters.
  • Predicting the energy consumption per output token based on the active parameter count of models not in the dataset (which may require educated guesses for unpublished data).
  • Multiplying the single GPU usage with the estimated number of active GPUs required, which is determined by the memory needed to run the model.
  • Summing this with the server energy usage without GPUs to obtain the total usage impact.

Embodied Impacts

These include resource extraction, hardware manufacturing, and transportation. The calculation involves:

  • Using BoaviztAPI to estimate the embodied impacts.
  • Allocating these impacts per request using a hardware utilization factor.
